# Session 000012 — Fix CSS/JS asset loading (design.md finalmente aplicado)

## Problem
O site estava renderizando sem CSS — o usuário reportou visual "Yahoo 1995", confirmando que o design "Universe Deep Space" do `design.md` nunca foi aplicado de fato.

## Root Cause
**Dois bugs concatenados:**

1. **`functions.php` nunca era carregado** — O `index.php` raiz carregava `wp-load.php` e depois incluía o tema, mas NUNCA incluía `wp-content/themes/tiranossaurusrex/functions.php`. Lá estava o `add_action('wp_enqueue_scripts', 'tiranossaurusrex_enqueue_scripts')` que chama `wp_enqueue_style()` e `wp_enqueue_script()`. Sem carregar functions.php, essas chamadas nunca ocorriam.

2. **`wp_head()` e `wp_footer()` vazios** — Eram placeholders que não executavam nada. Mesmo se as enqueues funcionassem, não havia código para *imprimir* os `<link>` e `<script>` tags.

## Fixes Applied

### 1. Asset Enqueue System (wp-load.php:263-350)
- Implementado `$wp_actions`/`$wp_filters` globais com `add_action()`, `do_action()`, `add_filter()`, `apply_filters()`
- Implementado `$wp_styles`/`$wp_scripts` globais com `wp_enqueue_style()`, `wp_enqueue_script()`
- `wp_print_styles()` — gera `<link rel="stylesheet">` tags
- `wp_print_footer_scripts()` — gera `<script>` tags para scripts com `in_footer=true`
- `wp_head()` agora chama `do_action('wp_enqueue_scripts')` + `wp_print_styles()`
- `wp_footer()` agora chama `wp_print_footer_scripts()`

### 2. Theme bootstrap (index.php raiz:5-8)
- Adicionado `require_once $functions_file` após carregar `wp-load.php`

### 3. Duplicate function removal (functions.php)
- Removido `tiranossaurusrex_pagination()` duplicado (já existia em wp-load.php)
- Removido `tiranossaurusrex_get_reading_time()` duplicado (já existia em wp-load.php)

### 4. `get_bloginfo()` melhorado (wp-load.php)
- Agora retorna 'description' e 'charset' além de 'name'

## Files Modified
- `wordpress/wp-load.php` — +90 linhas (asset system + action hooks)
- `wordpress/index.php` — +5 linhas (include functions.php)
- `wordpress/wp-content/themes/tiranossaurusrex/functions.php` — -19 linhas (remove duplicatas)

## Verification
- `<link rel="stylesheet" id="tiranossaurusrex-style-css">` presente no `<head>`
- `<script id="tiranossaurusrex-main-js">` presente antes de `</body>`
- `charset="UTF-8"` corrigido
- Zero PHP errors/warnings na página

O design "Universe Deep Space" com fundo preto, gradientes, glow e HUD style DEVE estar visível agora.
