## Work done

### Phase 1: OG Meta Tags
- header.php now outputs `og:title`, `og:description`, `og:image`, `og:url`, `og:type`, `og:site_name`, `og:locale`
- Also outputs `twitter:card` (summary_large_image), `twitter:title`, `twitter:description`, `twitter:image`
- On single posts: reads from `$GLOBALS['post']` (set by `setup_postdata()` before `get_header()`)
- On homepage: shows site default title/description/logo
- Description is full excerpt up to 350 chars (then truncated with "...") — NOT abbreviated
- Verified working: curl shows correct OG tags on both single post and homepage

### Phase 2: Database schema
Extended `trex_add_post()` with `$extra` parameter supporting:
- `post_type`: 'article' | 'video' | 'audio' | 'gallery'
- `youtube_url`: YouTube video link
- `audio_url`: audio file URL
- `gallery`: array of image URLs
- `social_shared`: object tracking which platforms were shared to

Updated `trex_update_post()` to handle the new fields specifically.

### Phase 3: Social media sharing engine
Added to wp-load.php:
- `trex_get_share_text($post)` — builds share message with title, lead (up to 150 chars), excerpt (up to 300 chars), URL, image
- `trex_share_to_telegram($post)` — sends photo+caption or message+link via Telegram Bot API
- `trex_share_to_facebook($post)` — posts link+message via Facebook Graph API (`/v19.0/me/feed`)
- `trex_share_to_instagram($post)` — creates media via Graph API, then publishes it
- `trex_share_to_tiktok($post)` — posts via TikTok Open API
- `trex_share_to_all($post_id)` — orchestrator that calls all 4 platforms, skips already-shared

### Phase 4: Admin interface
Added to admin.php:
- Social media config section with fields for Telegram Bot Token, Chat ID, Facebook Page Token, Instagram Token, Instagram ID, TikTok Token
- Saved via index.php POST handler with `update_option()` for each field
- API docs section showing the POST /api/v1/posts JSON schema and share endpoints

### Phase 5: API endpoints
Updated index.php REST handler:
- `POST /api/v1/posts` — now accepts `post_type`, `youtube_url`, `audio_url`, `gallery`, `share` ("auto" | "none")
- `GET /api/v1/share/{id}` — manually trigger sharing for existing post
- `GET /api/v1/options` — check status of all social media tokens
