# Session 000045 — Mobile/tablet: header full-width sem padding lateral

## Date
2026-06-28

## Task
Header do Tiranossaurus Rex deve ocupar toda a largura da tela em mobile e tablet, sem limitações nas laterais. A restrição de largura (`max-width: 1200px` com padding lateral) deve valer apenas para desktop.

## Root cause
`.header-wrapper` tinha `padding: 0 30px` em todas as telas (adicionado nas tentativas 6-9). Em mobile/tablet, isso criava um recuo de 30px em cada lado, impedindo o header de ocupar a largura total da tela.

## Changes made
Adicionado `@media (max-width: 768px) { .header-wrapper { padding: 0; } }` — remove o padding lateral do header em telas ≤768px, permitindo que o header ocupe toda a largura da tela em mobile e tablet.

### Desktop (≥768px) — mantém o comportamento existente:
- `.header-wrapper { padding: 0 30px; max-width: 1200px; margin: 0 auto; }`
- Header alinhado com `.site-content` (30px de padding)

### Mobile/Tablet (<768px) — novo comportamento:
- `.header-wrapper { padding: 0; }`
- Header ocupa 100% da largura da viewport
- O `.site-header` mantém seu próprio padding (12px no 768px, 8px no 480px) para espaçamento interno
