# Session 000054 — Facebook API dead end: Games apps don't have pages_manage_posts

## Date
2026-07-02

## Task
Postar automaticamente no Facebook Page do Blog Tiranossaurus Rex sempre que um novo post for criado.

## Resumo do problema
Após ~50 horas de tentativas, o erro central foi identificado: **apps do tipo "Games" não oferecem a permissão `pages_manage_posts`**. Ambos os apps criados pelo usuário (`1029129493002825` e `1791222012241978`) são do tipo Games (criados via `/games/?app_id=`). Por isso a permissão nunca aparecia no Graph API Explorer.

## Solução real identificada

1. developers.facebook.com > Criar App > Business
2. Adicionar permissões: pages_manage_posts, pages_read_engagement
3. Gerar token via Graph API Explorer
4. Usar token no .env como FACEBOOK_PAGE_TOKEN
