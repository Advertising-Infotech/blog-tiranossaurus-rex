# SESSION 000019 — Fix: logotipos ausentes, menu mobile, ads, paginação, foto

## Problemas Reportados

### 1. Logotipos não aparecendo
**Causa raiz**: Os 4 arquivos PNG (`Logotipo 1.png` ~ `Logotipo 4.png`) existiam em `wordpress/Logotipo/` mas NUNCA foram copiados para `wp-content/themes/tiranossaurusrex/images/`. O header.php referenciava caminhos em `images/` mas os arquivos não estavam lá.
**Fix**: Copiados os 4 PNGs de `Logotipo/` para `images/`.

### 2. Menu mobile não corrigido
**Causa**: `initScrollHide()` no `main.js` só escondia a nav (`<nav class="main-navigation">`), mas não o header.
**Fix**: Adicionado `header.classList.add('header-hidden')` no scroll down e CSS `.header-hidden` com `transform: translateY(-100%)`.

### 3. Espaço publicitário acima de "Categorias" sumiu
**Causa**: O `.sidebar-ad-square` existe no `sidebar.php` e o CSS está presente. Pode ter sido C:\trex que estava desatualizado.
**Fix**: C:\trex sincronizado com todos os templates.

### 4. Botão "Próximo" sobrepondo numeração no mobile
**Causa**: Botões de paginação muito grandes para a largura mobile.
**Fix**: Reduzido padding/font-size, adicionado `min-width: 32px`, `flex-wrap` com `gap: 4px`, e centralização.

### 5. Foto de perfil aparecendo arredondada
**Causa**: CSS `border-radius: 12px` existia, mas CSS pode não ter sido carregado.
**Fix**: Adicionado `!important` no `border-radius: 12px` da `.profile-photo` para garantir que não seja sobrescrito.

### 6. Cards AdSense com intervalo errado
**Causa**: `$ad_counter % 6 === 0` (a cada 6 posts).
**Fix**: Alterado para `$ad_counter % 5 === 0` — ad a cada 5 posts (6ª posição, 12ª, etc.).

### 7. Banner rotativo sem pause no hover
**Fix**: Adicionado `mouseenter`/`mouseleave` listeners no banner que pausam/retomam o ciclo.

## Arquivos Modificados
- `wp-content/themes/tiranossaurusrex/index.php` — AdSense counter fix
- `wp-content/themes/tiranossaurusrex/style.css` — profile-photo !important, header-hidden CSS, card glow border, pagination mobile
- `wp-content/themes/tiranossaurusrex/js/main.js` — scrollHide includes header, banner hover pause
- `images/` — 4 arquivos de logotipo adicionados
- C:\trex sincronizado
